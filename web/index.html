Wallet Generator

Generate Cosmos and Ethereum addresses offline

For Testing & Educational Purposes Only This tool should NOT be used for managing real funds. Use established wallet software for production use.
secp256k1: Standard Cosmos address (SHA256 + RIPEMD160)
eth_secp256k1: EVM-compatible chains like Injective, EVMOS (Keccak256)
Note: Public key mode can only derive the Cosmos address. Accepts both hex (66/130 chars) and base64 formats.
Select a mode and click "Generate Addresses"
