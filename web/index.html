Wallet Generator

Generate Cosmos (Bech32) and Ethereum addresses offline

For Testing & Educational Purposes Only This tool should NOT be used for managing real funds in production. Use established wallet software for production use.
Note: Public key mode can only derive the Cosmos address. Private key and Ethereum address cannot be derived from a public key.
secp256k1: Standard Cosmos address (SHA256 + RIPEMD160)
eth_secp256k1: EVM-compatible Cosmos chains like Injective, EVMOS (Keccak256, same as ETH)
Select a mode and click "Generate Addresses"
